**README**

Steps to run the BOOKING SYSTEM:

1.Open the terminal

2.Locate the directory

3.Type the command given below

make

Note: It will automatically compile all the files and make executable file'booking'

4.In the terminal type the command given below

./booking

5.The booking system will start running

6.Enter the inputs as per the instructions provided by the program.
